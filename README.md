# CloudMusic Pro Player

基于 Web 前端技术栈的网易云音乐增强版在线播放器。

## 功能特性

### 核心播放
- 播放/暂停、上一曲/下一曲
- 进度条拖拽、音量调节
- 循环模式切换（顺序/单曲/随机）
- 音质切换（标准/高清/无损）
- 歌曲下载

### 歌单管理
- 接入网易云 API 获取推荐歌单
- 支持歌单切换
- 搜索歌曲/歌手
- 排行榜支持

### 用户登录
- 二维码登录
- 手机验证码登录
- 显示用户信息与歌单数量

### 歌词功能
- 同步显示歌词
- 支持翻译歌词
- 点击歌词单句循环
- Shift+点击多行选中循环
- 循环弹窗控制（播放/暂停/重播）
- 0.5x-2.0x 倍速调节

### 背景自定义
- 歌曲封面作为背景
- 本地上传图片
- 亮度/模糊度/透明度调节

### 音频可视化
- 底部音乐节奏频谱展示

## 使用说明

### 1. 启动方式

由于使用了 ES6 模块和现代浏览器 API，建议使用本地服务器运行：

#### 方式一：使用 Python
```bash
# 进入项目目录
cd 音乐播放器

# 启动 HTTP 服务器
python -m http.server 8080
```

然后在浏览器访问 http://localhost:8080

#### 方式二：使用 Node.js
```bash
# 安装 http-server
npm install -g http-server

# 启动服务器
http-server -p 8080
```

#### 方式三：VSCode Live Server
- 安装 VSCode Live Server 扩展
- 右键 index.html 选择 "Open with Live Server"

### 2. 功能使用

#### 播放音乐
1. 页面加载后会自动获取推荐歌单
2. 点击左侧歌单列表切换歌单
3. 点击歌曲列表中的歌曲开始播放
4. 使用底部播放控制栏进行播放控制

#### 搜索
1. 在顶部搜索框输入关键词
2. 按回车或点击搜索按钮
3. 点击搜索结果播放

#### 登录
1. 点击右上角"登录"按钮
2. 选择二维码登录或手机号登录
3. 扫码或输入验证码完成登录

#### 歌词循环
1. 点击歌词面板中的任意歌词行
2. 弹出循环控制弹窗
3. 可调节播放倍速
4. Shift+点击可多行选中

#### 背景设置
1. 鼠标移入右侧悬浮栏展开面板
2. 选择背景来源（封面/本地上传）
3. 调节亮度、模糊度、透明度

## 技术说明

### API 说明

本项目使用网易云音乐增强版 API，默认使用 Vercel 公共接口。如遇接口失效，可自行部署：

- [网易云音乐增强版 API](https://github.com/neteasecloudmusicapienhanced/api-enhanced)

### 浏览器兼容性

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

需要支持 Web Audio API 和 ES6+。

## 注意事项

1. 部分歌曲可能因版权原因无法播放
2. API 接口可能存在频率限制
3. 建议登录后获得完整功能体验
4. 音频可视化需要用户交互后才能启动（浏览器策略）
