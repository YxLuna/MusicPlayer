# 网易云音乐API启动器

这是一个用于启动网易云音乐API服务的工具。

## 使用方法

### 方式一：直接运行
在终端中运行以下命令：
```bash
cd 音乐播放器/启动器
node start-api.js
```

### 方式二：创建快捷方式
1. 创建一个批处理文件 `启动API.bat`：
```batch
@echo off
cd /d "你的项目路径\音乐播放器\启动器"
node start-api.js
pause
```

2. 双击运行即可

## 功能特点

- 自动启动API服务
- 自动打开管理页面
- 支持自定义端口
- 端口配置保存（下次启动自动使用上次的端口）

## 界面说明

启动后会打开一个网页，你可以：
- 查看API服务状态
- 点击"打开API"直接访问API
- 点击"修改端口"更改API端口
- 点击"打开播放器"启动音乐播放器

## 配置说明

端口配置保存在 `config.json` 文件中，你可以直接编辑该文件来修改端口。
