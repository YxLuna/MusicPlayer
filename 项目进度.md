## 二、当前仍存在的问题

### 问题 1：背景效果控件仍然无效 ⚠️

**期望效果**：
- 上传本地背景图片后，通过调整「模糊度」「透明度」「毛玻璃」滑块
- 让前景元素（歌词面板、歌曲信息卡片）变得透明/模糊
- 从而能够透过前景元素看到后面的背景图片

**当前状态**：
- 已修改 `updateBackground()` 函数，将效果应用于前景元素
- 但调整滑块后仍然看不到效果变化

**可能原因**：
1. 前景元素的 CSS `background` 是渐变 `var(--gradient-card)`，可能被 `backgroundColor` 覆盖后失去渐变效果
2. `backdrop-filter` 在某些浏览器或元素上可能不生效
3. 需要检查元素是否真的应用了内联样式

**需要进一步调试**：
- [ ] 检查浏览器控制台是否有错误
- [ ] 检查 `updateBackground()` 是否被正确调用
- [ ] 确认前景元素的 backdrop-filter 是否生效
- [ ] 可能需要改变实现方式：用 rgba 渐变叠加而非 backdrop-filter

## 四、技术备忘

### 4.1 相关文件
- `index.html` - 主页面结构
- `css/style.css` - 合并后的樱花主题样式
- `js/ui.js` - UI 控制逻辑
- `js/player.js` - 播放器核心功能
- `js/api.js` - 网易云 API 接口

### 4.2 关键 CSS 变量
```css
--sakura-pink: #FFB7C5;      /* 樱花粉 */
--sakura-deep: #E88BA0;       /* 深樱 */
--candy-peach: #FFAB76;       /* 蜜桃橙 */
--candy-rose: #FF9AAF;        /* 玫瑰粉 */
--gradient-card: linear-gradient(145deg, rgba(255, 255, 255, 0.95), rgba(255, 240, 245, 0.9));
```

### 4.3 JS 中的背景效果函数
- `updateBackground()` - 更新前景元素的模糊度和透明度
- `toggleGlassEffect(enabled)` - 切换毛玻璃效果
- `handleBgUpload(e)` - 处理背景图片上传
- `applyCrop()` - 应用裁剪后的背景
