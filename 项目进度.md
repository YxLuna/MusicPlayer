**待验证**：

- [ ] 调整「毛玻璃」开关观察效果
- [ ] 如果仍无效，需要在浏览器控制台调试 `updateBackground()` 是否被调用

**技术备忘**：
```javascript
// 当前 updateBackground() 实现
updateBackground() {
    const rawOpacity = this.opacity?.value ?? 70;
    const minOpacity = 0.25; // UI 最低保持 25% 不透明
    const maxOpacity = 1.0;
    const uiOpacity = Math.min(maxOpacity, Math.max(minOpacity, rawOpacity / 100));
    const appContainer = document.querySelector('.app-container');
    if (appContainer) {
        appContainer.style.opacity = uiOpacity;
    }
}
```

**关键修复**：
- 添加了 `Math.max(minOpacity, ...)` 保护，确保 opacity 最低为 0.25（25%）
- 这样 UI 永远不会完全消失，即使滑块调到 0，`.app-container` 也会保持 25% 不透明
