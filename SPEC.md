# 网易云音乐增强版播放器 - 项目规范

## 1. 项目概述

- **项目名称**: CloudMusic Pro Player
- **项目类型**: Web单页应用（SPA）
- **核心功能**: 基于网易云音乐增强API的在线音乐播放器，支持播放控制、歌单管理、歌词同步、背景自定义等
- **目标用户**: 音乐爱好者、动漫风格UI爱好者

## 2. UI/UX 规范

## 3. 功能规范

### 3.1 核心播放功能

- **播放/暂停**: 点击切换，带过渡动画
- **上一曲/下一曲**: 循环切换歌单内歌曲
- **进度条**:
  - 点击跳转
  - 拖拽实时更新
  - 显示已播放/总时长
- **音量调节**:
  - 滑块控制 0-100%
  - 点击图标静音/取消静音
- **循环模式**:
  - 顺序循环 → 单曲循环 → 随机播放
- **音质切换**: MP3(128k) / AAC(192k) / FLAC(320k) / WAV(无损)

### 3.2 歌曲管理

- **歌单获取**: 登录后获取用户歌单
- **默认歌单**: 推荐歌单/热歌榜（未登录时）
- **歌单切换**: 点击切换当前播放列表
- **歌曲列表**: 显示序号、封面、标题、歌手、时长
- **歌曲下载**: 调用API下载歌曲文件
- **搜索功能**:
  - 搜索歌曲/歌手/歌单
  - 实时搜索建议
  - 搜索结果展示

### 3.3 用户登录

- **二维码登录**: 生成登录二维码
- **手机验证码**: 输入手机号获取验证码登录
- **登录状态**: 显示用户头像、昵称、歌单数量
- **退出登录**: 清除登录状态

### 3.4 歌词功能

- **LRC解析**: 解析时间轴歌词
- **同步显示**: 高亮当前播放行
- **翻译歌词**: 显示中文翻译（如有）
- **罗马音**: 显示罗马音标注（如有）
- **点击循环**:
  - 点击单句循环播放该句
  - Shift+点击多行选中范围
  - 弹出循环控制弹窗
- **循环控制弹窗**:
  - 显示选中的歌词内容
  - 显示时间范围
  - 播放/暂停按钮
  - 重播按钮
  - 0.5x-2.0x倍速选择

### 3.5 背景自定义

- **背景来源**:
  - 歌曲封面（默认）
  - 本地上传图片
  - 本地上传视频
- **图片调节**:
  - 缩放: 50% - 200%
  - 旋转: 0 - 360度
  - 裁剪: 宽高比选择
- **效果调节**:
  - 亮度: 0% - 200%
  - 模糊度: 0 - 30px
  - 透明度: 0% - 100%
- **实时预览**: 所有调节即时生效

### 3.6 音频可视化

- **实现方式**: Canvas绑定AudioContext
- **显示位置**: 播放控制栏底部
- **样式**: 柱状频谱，渐变色

### 3.7 状态反馈

- **加载状态**: 骨架屏/加载动画
- **操作反馈**: Toast提示
- **错误处理**: 错误弹窗/自动重试
- **播放状态**: 底部指示器

## 4. 技术实现

### 4.1 技术栈

- **HTML5**: 语义化标签
- **CSS3**: Flexbox/Grid/动画/滤镜
- **JavaScript**: ES6+ / 模块化
- **API**: 网易云音乐增强版API
- **构建**: 原生JS（无框架）

### 4.2 核心API调用

```
基础URL: https://netease-cloud-music-api-five-roan-25.vercel.app 或自建服务端

接口列表:
- /login/qr/key - 获取二维码key
- /login/qr/create - 生成二维码
- /login/qr/check - 检查二维码状态
- /login/cellphone - 手机号登录
- /user/account - 获取用户信息
- /user/playlist - 获取用户歌单
- /playlist/detail - 获取歌单详情
- /song/url - 获取歌曲URL
- /song/detail - 获取歌曲详情
- /lyric - 获取歌词
- /search - 搜索
```
