### 修改详情

1. **修复搜索功能报错** (`ui.js`)
   - 问题：`this.handleSearch is not a function`
   - 解决：将 `this.handleSearch()` 改为 `this.onSearch?.()` 回调函数

2. **修复搜索API数据结构** (`app.js`, `playlist.js`)
   - 问题：搜索API返回的数据结构是 `result.result.songs`
   - 解决：
     - `app.js`：条件判断改为 `result && result.result && result.result.songs`
     - `playlist.js`：`parseSearchResult` 方法改为访问 `searchResult?.songs`

3. **主界面左右分布布局** (`index.html`, `style.css`)
   - 整体容器改为 `.main-display`（flex左右布局）
   - 左侧 `.display-left`：大封面+旋转唱片效果（320x320px）
   - 右侧 `.display-right`：歌曲信息卡片 + 歌词面板
   - 保留视觉效果：毛玻璃、过渡动画、黑胶唱片旋转

4. **完善搜索结果显示** (`ui.js`, `style.css`)
   - 添加专辑名显示：`${song.artist} · ${song.album}`
   - 添加时长显示：`.search-result-duration`
   - 封面尺寸改为 48x48px
   - 添加 flex 布局优化空间利用

5. **修复搜索结果渲染问题** (`playlist.js`, `ui.js`)
   - 问题：搜索结果无法显示封面、歌手名、专辑名
   - 原因：搜索API返回字段名为 `artists`/`album`，与歌单详情的 `ar`/`al` 不同
   - 解决：
     - `parseSearchResult` 方法调用歌曲详情API获取完整信息
     - `formatSong` 方法增强字段兼容处理

6. **毛玻璃效果开关** (`index.html`, `style.css`, `ui.js`)

   - 支持切换：歌词面板、歌曲信息卡片、右侧面板、底部播放器等区域的毛玻璃效果

7. **背景图片持久保存** (`ui.js`)

   - 刷新页面后自动恢复背景设置

## 使用说明

### 启动方式（推荐：使用启动器）

#### 使用启动器一键启动（推荐）

双击运行 `启动器/启动API.bat` 即可自动启动：
- 网易云API服务（默认端口3000）
- 音乐播放器服务（默认端口5500）
- 启动器管理页面（端口3001，自动打开浏览器）

#### 手动启动（可选）

##### 1. 启动 API 服务（本地 3000 端口）

```bash
# 进入 API 项目目录
cd D:\桌面\api-enhanced-main

# 启动 API 服务
node app.js
```

##### 2. 启动前端服务

```bash
cd 音乐播放器
python -m http.server 5500
```

访问 http://localhost:5500

> **注意**: 需要先启动 3000 端口的 API 服务，前端才能正常工作

### 启动器配置

- 配置文件：`启动器/config.json`
- 可修改端口：API端口、播放器端口
- 修改端口后自动重启服务

#### 3. 歌词滚动位置计算修复

| 文件 | 修改内容 |
|------|---------|
| ui.js | 修改 `highlightLyric()` 使用 getBoundingClientRect() 精确计算滚动位置 |
| ui.js | 修改 `scrollToCurrentLyric()` 同样使用精确计算 |

**问题原因**: 之前的计算只减去容器高度的一半，导致歌词偏下
**解决方案**: 使用 `getBoundingClientRect()` 获取视口中的精确位置，让当前歌词在面板正中央显示
